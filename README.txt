For gameplay video see
https://www.youtube.com/watch?v=ht9LJn0BZRs

For downloadable game see
https://www.dropbox.com/s/p9ao0ghi0qd4u18/ecce.zip?dl=0

Open the folder and run EcceSimulacrum.exe

Troubleshooting:
If the game doesn't start, try installing the VC++ 2015 redistributable from
https://www.microsoft.com/en-us/download/details.aspx?id=53840

If there's no audio while playing the game, try installing OpenAL (1.1 Windows Installer) from
https://www.openal.org/downloads/
